# Old School Tracker

A standalone browser-based ProTracker MOD player and a tribute to the music of
Matthew "4mat" Simmonds.

The player originated as a hidden tracker screen in The Big (Web) Demo and was
later separated into this standalone project.

## Run locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The production build is written to `dist/` and uses relative asset paths, so it
can be hosted from a domain root or a subdirectory.

## Controls

- Click the opening banner to activate playback and keyboard controls.
- Click a MOD title to load and play it.
- Use the transport buttons for previous, play/pause, stop, and next.
- Use `Space` to play or pause.
- Use `Up` and `Down` to select a MOD, then `Enter` to load it.
- Use `Left` and `Right` to jump backward or forward by 10 MODs.
- Select `SCOPE`, `BARS`, `VU4`, or `RASTER` to change the visualizer.

MOD files and their display names are defined in
`public/audio/manifest.json`. Backdrop images are defined in
`public/images/manifest.json`.
